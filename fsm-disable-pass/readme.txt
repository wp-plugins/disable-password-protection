This plugin will remove the "Password Protect this post" from the writing screen so users cannot password protect the post.

== Description ==

Blogs that have multiple authors may want to completely disable authors or contributors from being able to
password protect a post.

This plugin will completely remove the "Password Protect this post" from the writing screen so users are not
even given the option.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `fsm-disable-pass.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!  You're done and ready to keep posts password-free!

== Frequently Asked Questions ==

= What if I want to enable passwords for a post? =

If you want to enable passwords for a single post, you will need to disable this plugin.
